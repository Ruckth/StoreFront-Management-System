# Leather Product Test Asset Pack

This folder contains local-only leather product test assets for StoreFront Management System manual testing.

- `images/` contains logo-free generated PNG catalog images sized to fit the StoreFront product UI.
- `products.json` contains matching product metadata, prices, descriptions, quantities, and relative image paths.
- These assets are not wired into the app, backend database, or existing frontend demo catalog.

The generated originals remain in the Codex image generation output folder; these files are project-local copies with stable names.
